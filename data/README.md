# data フォルダ

ここに以下の CSV ファイルを配置してください。

| ファイル名    | 内容                         | 必須/任意 |
|--------------|------------------------------|---------|
| topix.csv    | TOPIX 日次データ              | 必須    |
| sp500.csv    | S&P 500 日次データ（USD建て） | 必須    |
| gold.csv     | 金スポット価格（USD建て）      | 必須    |
| usdjpy.csv   | USD/JPY 為替レート            | 必須    |
| jgb.csv      | 日本国債1年利回り（%）         | 任意    |

## CSV フォーマット

ヘッダー行不要。1列目=日付、2列目=値。

```
2000-01-04,1722.56
2000-01-05,1708.33
```

詳細は README.md を参照してください。

> **注意:** データファイル自体はリポジトリに含まれていません。
> 上記の取得先からダウンロードして配置してください。
